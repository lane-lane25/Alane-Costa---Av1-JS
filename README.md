# 📝 Lista de Tarefas

Uma aplicação web interativa para gerenciar suas tarefas do dia a dia com uma interface limpa e intuitiva.

## ✨ Recursos

- ✅ **Adicionar tarefas** - Crie novas tarefas digitando e clicando em "Adicionar"
- ✏️ **Editar tarefas** - Modifique suas tarefas existentes usando o botão "Editar"
- 🗑️ **Excluir tarefas** - Remova tarefas que não precisa mais
- ✔️ **Marcar como concluída** - Clique na tarefa para marcá-la como concluída
- 🎨 **Interface responsiva** - Design modernizado com cores pastéis

## 🎯 Como Usar

1. Digite sua tarefa no campo de entrada
2. Clique em "Adicionar" para criar a tarefa
3. Use os botões para:
   - **Editar**: Modificar o texto da tarefa
   - **Excluir**: Remover a tarefa permanentemente
   - Clique na tarefa para marcar/desmarcar como concluída

## 📋 Validações

- ⚠️ Não são permitidas tarefas vazias
- ⚠️ Não é possível adicionar tarefas duplicadas

## 🌸 Easter Egg

Se você digitar "Jane Austen" em uma tarefa, flores caíram na tela! 🌸
